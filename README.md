# SkillMatch JS Simulator

SkillMatch JS Simulator is a job listing and compatibility assessment tool that calculates the percentage match between candidates and available job opportunities, making it possible to identify which position best fits the skills of the candidate during the registration process.
  
## Can you explain the purpose of this project?

## How do I run this JavaScript file?
1. Abra qualquer site no navegador
2. Copie o código que está no arquivo skillmatch.js desse repositório que você está atualmente
3. Abra o console através do atalho de teclado F12 ou procure pelas ferramentas no menu de configuração e depois ferramentas do desenvolvedor
4. Vá até a aba Console nas ferramentas de desenvolvedor e cole o código
5. Clique no final da última linha do código e dê um 'enter' pelo teclado
6. O navegador abrirá a interação com o programa através de popups
7. Divirta-se!
